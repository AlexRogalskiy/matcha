☑️ Supported browsers

matcha.css is designed to support all modern browsers. See below for more details.

Methodology:
  1. Generate matcha.css stylesheet.
  2. Generate report table using jsr.io/@libs/bundle, which does the following:
    1. Get browsers list using browserslist.
    2. Parse stylesheet using csstree.
    3. Check support of each CSS features over each browser version using MDN browser compatibility data.
    4. Generate a <table> summary
Notes:
  • Browser list is set to defaults (> 0.5%, last 2 versions, Firefox ESR, not dead).
  • The first row display the overall support for a given browser column.
  • As the table is auto-generated and it is hard to map back every CSS features against compatibility data, so it may contain inaccuracies.
  • Thus, a browser with a score of 90% and above is considered supported.
